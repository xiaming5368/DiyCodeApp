• Diycode是一个什么样的社区?
  • 我一直不知道应该给开发工程师搭建一个什么样的社区，也不知道应该如何定义开发工程师这群人，正如雷军说的编程不仅仅是技术，还是艺术，我也一直认为开发工程师不仅仅只是写代码的人，我们是真正的创造者，所以我想象中diycode交流的话题可以是关于DIY Code 的艺术，也可以是关于创业的点子，更可以是分享发现所有有趣事物，我希望可以通过建立一套社区规则，让整个社区的气质始终保持在优雅、极客、创意、分享、美好的氛围里。
